---------------------------------------------------------------------
ビルドには外部ソフトウェアが必要です。
---------------------------------------------------------------------
必要なソフトウェア
・CMAKE
https://cmake.org/
・CONAN
https://conan.io/
・MSVC
https://learn.microsoft.com/ja-jp/cpp/overview/compiler-versions?view=msvc-170


依存するライブラリ
・filament
https://github.com/google/filament/releases
・physx
https://github.com/NVIDIA-Omniverse/PhysX

依存するライブラリのビルド済みデータの導入方法
・ovphysx-windows-x86_64~.zip
・filament-~-windows.tgz

filament-~-windows->filament
ovphysx-windows-x86_64~->physx

のようにbuild.batなおるフォルダーに入れます。

リリースビルドの場合
・Release.batを使用してください。
デバッグビルドの場合
・Build.batを使用してください。
